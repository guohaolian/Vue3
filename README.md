# Vue3

这是一个基于 Vue 3 + Vite + TypeScript 的示例项目。

## 项目简介

- 前端框架：Vue 3
- 构建工具：Vite
- 开发语言：TypeScript
- 适合用于学习 Vue 3、组件开发和前端工程化实践

## 快速开始

```sh
npm install
npm run dev
```

## 构建生产版本

```sh
npm run build
```
